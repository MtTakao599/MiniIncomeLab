# Linux向けミニPCの選び方

Linux をメイン OS として使う場合、ハードウェア互換性とドライバの情報量が重要です。Ubuntu や Debian など、使うディストリビューションに合わせて選ぶのが安全です。

## 向いている人

- サーバー検証や開発環境を自宅で持ちたい
- Windows ライセンスを別途抑えたい
- CLI 中心の用途が多い

## チェックポイント

- 内蔵 Wi-Fi / Bluetooth の Linux 対応情報を調べる
- UEFI 起動と Secure Boot の扱いを確認
- メモリ増設可否と上限を確認

## 注意点

- 事前に同型機の Linux 導入事例があると安心
- 公式サポート外 OS の利用は自己責任で行う
